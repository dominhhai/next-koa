# next-koa
An introduction to [Koa v2](https://github.com/koajs/koa) - the next generation web framework for Node.js with Power of ES6, ES7 (async-await)

![next-koa](/next-koa.PNG)

### Installation

```
$ npm i -g next-koa
```

Permission Fail:

```
$ sudo npm i -g next-koa
``` 

### Usage

```
$ next-koa
```
